# 블록체인을 이용한 채용 점수 공개

## 문제점
수많은 기업에서 채용 시 점수 조작, 낙하산 채용 등 공정하지 못한 방법으로 인해 적절한 인재의 채용이 진행되지 않는 사건들이 발생한다.

## 해결책
서류, 1차, 2차, 최종면접 등 각각의 채점결과를 블록체인으로 구성한다.
후에 수정이 불가능하기 때문에 구직자 입장에서도 실제 자신의 역량만큼을 적을 수 있고 기업 입장에서도 구직자를 채용한 뒤 실망한 일이 없을 것이다.

## benefit
- 기업
-- 기업 이미지 향상
- 구직자
-- 정확한 정보 획득
- 지원자
-- 합/불 정확한 이유를 알 수 있음
